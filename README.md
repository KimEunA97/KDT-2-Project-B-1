## 프로젝트 명 : 먹보의 여행

## 프로젝트 목표

지도를 기반으로, '먹보' 라는 가상의 인물을 상정 -> 맛집을 추천해주는 서비스

## 구현 기능

1. 지도위에 마커를 커스텀하여 클릭했을때 데이터 크롤링을 통해 가져온 맛집 리스트 구현
2. 카카오API를 통한 현재위치에서 부터 맛집까지의 경로를 알수있는 폴리라인 구현
3. 네비게이션 화면 구현
4. 맛집 인근 주차장 정보 구현

## 사용한 기술

- Reactnative
- nest.js
- kakao API
- google API
